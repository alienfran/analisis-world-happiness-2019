# Análisis de Datos — World Happiness Report 2019

## Descripción
Análisis exploratorio del World Happiness Report 2019, realizado como proyecto 
integrador de la Fase 1 de mi ruta de aprendizaje en análisis de datos.

## Herramientas utilizadas
- Google Sheets
- Funciones: PROMEDIO, MEDIANA, DESVEST, CONTAR.SI, SUMAR.SI
- Tablas dinámicas
- Histogramas

## Proceso
1. Clasificación de tipos de datos
2. Limpieza de datos (corrección de errores de formato decimal)
3. Estadística descriptiva del puntaje de felicidad
4. Clasificación de países por nivel de felicidad (Alto/Medio/Bajo)
5. Tabla dinámica con promedio de felicidad por categoría
6. Histograma de distribución

## Hallazgos principales
- El puntaje promedio mundial de felicidad es 5.407
- 88 de 156 países se ubican en el rango medio de felicidad
- La brecha entre países "Alto" (6.645) y "Bajo" (3.510) supera los 3 puntos
- La distribución del puntaje se aproxima a una distribución normal
